# Python Stuff

A collection of basic Python scripts that demonstrate core programming concepts.

## Included Scripts

- **`lens_slice.py`**: Works with Python lists (slicing, 2D lists, and counting).
- **`p2.py`**: A simple interactive command-line application for managing a daily to-do list.
- **`p4.py`**: Demonstrates logic for calculating averages, revenues, and using list comprehensions for a hair salon.
- **`hello.py`**: Basic string multiplication and variable assignments.
- **`Iam.py`**: Identity and Access Management simulation script for user tool assignment based on roles.
- **`DiskDriver.py`**: A disk space calculator that alerts the user if space is running out.

## Usage

You can run any of the scripts using Python:
```bash
python <script_name>.py
```
