Carly's Clippers Data Analysis

This project is a component of the Codecademy Learn Python 3 course. It provides a Python-based analysis tool for Carly's Clippers to evaluate revenue, haircut pricing, and marketing opportunities.
Project Overview

The script processes salon data to perform the following operations:

    Calculate the average price of all services.

    Apply a price adjustment to all existing service costs.

    Compute the total revenue generated based on weekly sales volume.

    Identify and filter specific services that fall under a target price point for promotional purposes.

Requirements

    Python 3.x

File Structure

    p4.py: The main analysis script containing the logic for data processing.

How to Run

    Ensure you have Python installed on your system.

    Open your terminal or command prompt.

    Navigate to the directory containing the file.

    Execute the script using the following command:
    Bash

    python p4.py

Key Features

    Revenue Tracking: Calculates total income by mapping service prices to their respective sales counts.

    Price Optimization: Utilizes list comprehension for efficient data transformation.

    Targeted Marketing: Dynamically filters services to highlight affordable options for customers.
